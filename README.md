# 🌧️ Building SafeStreet — Interactive Showcase & Story

Welcome to the documentation and showcase repository for **SafeStreet**, an AI-powered, hyper-local flood risk advisor built for Mumbai's monsoon commuters. 

This repository hosts the deployed interactive blog webpage that breaks down the technical architecture, problem statement, Gemini AI tool-calling setup, and real-time prototype media demo.

### 📑 Article Overview & Structure

The showcase covers the complete end-to-end journey of building SafeStreet:

* **01. The Problem:** Why broad weather warnings ("Heavy rain in Mumbai") fail daily commuters on the ground.
* **02. Problem Statement:** Bridging the gap between city-wide satellite alerts and street-level waterlogging reality.
* **03. Core Solution:** How SafeStreet calculates hyper-local risk scores using rain duration, ward drainage baselines, live news sentiment, and crowdsourced reports.
* **04. Feature Breakdown:** Detailed architectural decisions—from Open-Meteo hourly history tracking to Gemini 2.5 Flash ADK tool calling.
* **05. Tech Stack:** Why FastAPI, Google Cloud ADK, Gemini 2.5 Flash, Firestore, and Docker were chosen for real-world reliability.
* **06. Interactive Prototype Showcase:** Built-in Swiper.js media carousel displaying short video walkthroughs and UI snapshots.
* **07. Engineering Lessons:** Handling API rate limits, failing gracefully, and maintaining honesty in public safety tools.
